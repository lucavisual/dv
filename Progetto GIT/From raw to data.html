Short Description

The flowchart below shows all the steps we have done to modify the original dataset to get the one from which we created our views. We made many changes because the original dataset included incorrect or incomplete data.





1. We download the dataset from kaggle.com

2. The dataset contained too less information of the countries out of Africa so We decided to focus on African country

3. We increased dataset with other informations taken by data.humandata

3. Some nations were misssed from the dataset so we got their informations from other dataset and we joined all

4. We deleted some column of the database that was useless for us

5. We want to show the coordinats of African countries so with a code we provide all contry's latidude and longitude

6. We wanted to compare the food price so we converted all values to dollar
